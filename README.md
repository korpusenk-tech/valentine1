# Valentine's Day Website

Сайт ко Дню святого Валентина.

## Деплой на Railway

1. Установи Railway CLI (если еще не установлен):
   ```bash
   npm i -g @railway/cli
   ```

2. Войди в Railway:
   ```bash
   railway login
   ```

3. Инициализируй проект:
   ```bash
   railway init
   ```

4. Задеплой:
   ```bash
   railway up
   ```

Или используй веб-интерфейс Railway:
1. Зайди на https://railway.app
2. Создай новый проект
3. Подключи GitHub репозиторий или загрузи файлы
4. Railway автоматически определит Node.js проект и задеплоит
